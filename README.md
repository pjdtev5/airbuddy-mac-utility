## AirBuddy 2 是什么，一句话说清楚

AirBuddy 2 是一款 macOS 专属的蓝牙外设增强工具，由独立开发者 Gui Rambo 开发。简单粗暴地说，它做的事就一件：**把你在 iPhone 上用 AirPods 时那些"理所当然"的体验，完整搬到 Mac 上**。

它常驻在菜单栏，要求 macOS 10.14.6 或更高版本，并且你的 Mac 需要支持蓝牙低功耗（BLE）。Apple Silicon 原生适配，M 系芯片跑起来没有任何别扭。

## 几个核心功能，每一个都解决具体痛点

**开盒弹窗动画**：掀开 AirPods 盒盖，Mac 屏幕立刻弹出熟悉的连接动画——左右耳塞电量、耳机盒电量，一目了然。第一次看到会有种"哦，这才对嘛"的感觉。不用再点蓝牙菜单，不用进设置。

**菜单栏外设总览**：点击菜单栏图标，周围所有苹果外设的状态一屏看尽——AirPods、Magic Mouse、Magic Keyboard、Magic Trackpad、Apple Watch、iPhone、iPad 的电量都能显示。设备按层级分组，比如 Apple Watch 会挂在 iPhone 下面，符合直觉。

**快捷键切换降噪/通透模式**：自定义键盘快捷键，一秒在降噪、通透、Normal 三种模式之间切换。开视频会议想听到同事声音？按一个键就行，不用伸手去捏耳机柄。

**Magic Handoff 设备接力**：有两台 Mac 的话，这个功能会让你感动。右键点击 AirBuddy 菜单里的键鼠条目，一键把 Magic Mouse 或 Magic Keyboard 从当前 Mac 接力给另一台，省掉手动切蓝牙配对的麻烦。双机党会懂这有多省时。

**电量预警弹窗**：设一个电量阈值，低于这个值就弹窗提醒；充满电也提醒。阈值自己定，防止过充或者耳机突然没电。设置会在你所有装了 AirBuddy 的 Mac 之间同步。

**快捷指令（Shortcuts）集成**：AirBuddy 内置了 Shortcuts App 动作，可以查询设备电量、连接/断开 AirPods、管理连接模式，2.6 版本之后还加入了空间音频（Spatial Audio）控制动作，可以配合自动化场景玩出花样。

**AirPods 小组件**：2.6 版本新增的小组件，配合 macOS 的桌面组件功能，把耳机电量钉在桌面上，扫一眼就知道还剩多少。
